# HR App
 - Portal for HR


### CLone The Repo
    $ git clone  https://github.com/mahasagar/hr_app.git
    $ cd hr_app

### Install Dependency
    $ npm install
    $ bower install

### How to Run
    $ nodemon server.js   #Local

### How to Deploy
    $ pm2 start server.js  #Production


### Tech Stack
    - Angularjs : 1.6.5
    - Nodejs : 8.9.0
    - MongoDB : 3.4
